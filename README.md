# Tailwind Cloning Gate

This project is a **UI cloning assessment** completed during my training at **The Gym**.

The goal of the gate was to **replicate the layout of a real website using TailwindCSS**, focusing on layout accuracy, responsiveness, and styling details.

The page structure and styling were written **manually from scratch without using UI frameworks or generators**.

---

## What This Project Includes

- Responsive webpage layout
- Navigation bar with menu and icons
- Hero section with background image and overlay
- News section with grid layout
- Event cards with hover effects
- Image sections with overlay content
- Horizontal scrolling cards
- Footer with multiple information sections
- Social media and contact links

---

## Technologies Used

- **HTML5**
- **TailwindCSS**
- **Font Awesome Icons**
- **Google Fonts**
